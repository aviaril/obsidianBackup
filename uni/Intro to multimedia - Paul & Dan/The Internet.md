### Internet history
- originally developed by U.S DoD
	- development started in 1969 
	- enabled communication between geographically dispersed research centres
- ARPAnet was the first WAN
- Electronic mail was introduced using SMTP
- in the early 80's the current versions of TCP and IP were introduced 
- in the late 80's DNS was introduced 
- in 93 CERN released first version of WWW software

### Key Properties 
- Interoperable - any computer connected to the internet can communicate with any other connected computer 
- Global - based on standardised and universal connectivity 
- Easy to use 
- Low cost
	- cost is shared across all users 
	- an information provider does not need to pay for distribution just its own connection 
#### Uniform Naming 
- the IP layer offers a uniform 32bit addressing structure (IPv4) 
- every device connected to the internet must have an IP address, including hosts and servers 
	- there are $256^4$ ~ 4.3 billion addresses which is not enough 
- IPv6 deals with IPv4 running out of addresses 
	- 128bit address instead of 32 $3.4\times 10^{38}$ addresses 
	- represented as eight groups of four hex digits separated by colons 
		- 2001:0db8:0000:0000:0000:8a2e:0370:7334
