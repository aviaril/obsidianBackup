- Type of breadth first search 

1. Set initial distance from start for all nodes 
2. For each node 
	1. calculate distance from start 
	2. if the distance from the start is lower than the currently recorded distance 
		1. set the shortest distance to the start of the connected node to the newly calculated distance 
		2. set previous example to be the current node 
3. mark node as visited
