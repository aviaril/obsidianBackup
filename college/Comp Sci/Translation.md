## Interpreter 
- converts source code line by line in at runtime 
- High Level languages
- Access to source code is available 
- Portable
- reports one error then stops 
- Program runs slower than compiled 
- Runs in VM 
- interpreter is needed every time the program is run 
## Compiler 
- converts all code before running
- High level languages
- Creates an executable that will run faster than interpreted code
- Architecture specific
- Compiler is no longer needed after compilation
- reports all errors at end of compilation
## Assembler 
- Converts assembly into machine code 
- low level languages
