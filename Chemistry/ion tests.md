### Sodium Hydroxide tests 

| Metal Ion            | Colour of Precipitate                              | Ionic Reaction                                                  |
| -------------------- | -------------------------------------------------- | --------------------------------------------------------------- |
| Silver Ag$^+$        | Brown (silver oxide)                               | Ag$^{+}_{(aq)}$ + 2OH$_{aq}$ → Ag$_2$O$_{(s)}$ + H$_2$O$_{(l)}$ |
| Calcium Ca$^{2+}$    | White                                              | Ca$^{2+}_{aq}$ + 2OH$_{aq}$ → Ca(OH)$_{2(s)}$                   |
| Copper(II) Cu$^{2+}$ | Blue                                               | Cu$^{2+}$ + 2OH$_{aq}$ → Cu(OH)$_{2(s)}$                        |
| Lead(II) Pb$^{2+}$   | White                                              | Pb$^{2+}$ + 2OH$_{aq}$ → Pb(OH)$_{2(s)}$                        |
| Iron(II) Fe$^{2+}$   | Green                                              | Fe$^{2+}$ + 2OH$_{aq}$ → Fe(OH)$_{2(s)}$                        |
| Iron(III) Fe$^{3+}$  | Reddish Brown                                      | Fe$^{3+}$ + 3OH$_{aq}$ → Fe(OH)$_{3(s)}$                        |
| Zinc Zn$^{2+}$       | White, redissolves in excess NaOH turns colourless | Zn$^{2+}$ + 2OH$^-$ → Zn(OH)$_{2(s)}$                           |
| Aluminium Al$^{3+}$  | White, redissolves in excess NaOH turns colourless | Al$^{3+}$ + 3OH$^-$ → Al(OH)$_{3(s)}$                           |
### Silver Nitrate Halide tests 
Add dilute nitric acid and then silver nitrate to a sample 

##### Chloride 
White precipitate 
Ag$^+$ + Cl$^-$ → AgCl
##### Bromide 
Cream Precipitate 
Ag$^+$ + Br$^-$ → AgBr
##### Iodide 
Yellow Precipitate 
Ag$^+$ + I$^-$ → AgI
